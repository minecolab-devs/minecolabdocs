## Change Server Software

<img src="/assets/images/celdas/change_server.png" alt="change_server" />

Esta celda permite al usuario **cambiar el tipo de servidor Minecraft** que se está ejecutando en Colab.

:::caution
Cambiar el software del servidor **eliminará el servidor actual**.
:::

**Recomendaciones antes de ejecutar la celda:**

* Haz un **respaldo de tus mundos y configuraciones** si deseas conservarlos.
* Verifica que **no haya jugadores conectados** al servidor antes de continuar.

:::note
La configuración es la misma que ya se habló en [Getting started](/minecolab/guide/getting-started/getting-started) y [Tunnels](/minecolab/guide/tunnel/tunnels).
:::

## Safely Change Server Software

<img src="/assets/images/celdas/safely_server.png" alt="safely_server" />

Esta celda permite al usuario **cambiar o actualizar la versión del servidor Minecraft** que se está ejecutando en Colab, **sin eliminar el servidor actual**.

**Recomendaciones antes de ejecutar la celda:**

* Haz un **respaldo de tus mundos y configuraciones** por seguridad.
* Verifica que **no haya jugadores conectados** al servidor antes de continuar.

:::note
La configuración es la misma que ya se habló en
[Getting started](/minecolab/guide/getting-started/getting-started) y [Tunnels](/minecolab/guide/tunnel/tunnels)
:::
