## Cómo Cambiar el Servicio de Túnel

<img src="/assets/images/celdas/tunnels_change.png" alt="tunnels_change" />

1. Asegúrate de haber ejecutado al menos una vez tu servidor para inicializar la configuración.  
2. Abre la celda **Change Tunnel Service**.  
3. Selecciona el servicio de túnel que deseas usar en **`tunnel_service`**:  
   - `"Leave as it is"` → Mantiene el túnel actual.  
   - `"ngrok"`, `"argo"`, `"zrok"`, `"playit"`, `"localtonet"`, `"tailscale"`, `"minekube-gate"`  
4. Guarda los cambios; el script actualizará automáticamente la configuración del túnel.  
5. Reinicia el servidor si es necesario para activar el nuevo túnel.

## Finalización

¡Listo! Tu servidor de Minecraft ahora usa el nuevo túnel.  

- Verifica que el túnel esté activo antes de compartir tu servidor.  
- Si algo no funciona, vuelve a la celda **Change Tunnel Service** y selecciona otro túnel.  

Tu servidor está listo para jugar y compartir.
