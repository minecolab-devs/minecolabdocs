## Tunnels

Existen varios servicios de tunnel que te permiten exponer tu servidor de Minecraft en línea sin necesidad de abrir puertos manualmente.
Cada uno tiene sus propias características, limitaciones y beneficios.

## Tunnels disponibles

- **Ngrok**
- **PlayIt.gg**
- **Zrok**
- **LocalToNet**
- **Tailscale**
- **Minekube Gate**

---

## Limitaciones comunes

Cada proveedor de tunnel tiene restricciones que deberías considerar antes de elegir.

- **Ngrok**  
  El dominio cambia cada vez que reinicias (a menos que uses un plan de pago).
  Tiene límites en las conexiones simultáneas y el ancho de banda.
  Requiere una tarjeta de crédito para usar tu token.

- **PlayIt.gg**  
  Asigna una IP pública que normalmente se mantiene igual mientras se conserve la misma configuración.
  Algunas regiones pueden experimentar una latencia moderada.

- **Zrok**  
  Puede requerir configuración adicional.
  No está completamente optimizado para juegos en tiempo real.
  Algunas regiones pueden experimentar una latencia moderada.

- **LocalToNet**  
  El plan gratuito tiene límites de sesión.
  Algunas regiones pueden experimentar una latencia moderada.

- **Tailscale**  
  Requiere instalación del cliente y autenticación.
  Funciona mejor en entornos de red privada.
  Algunas regiones pueden experimentar una latencia moderada.

- **Minekube Gate**  
  Aún está en desarrollo.
  Puede no ser estable en todos los entornos.
  Algunas regiones pueden experimentar una latencia moderada.
