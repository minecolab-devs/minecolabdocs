## Change Server Software

<img src="/assets/images/celdas/change_server.png" alt="change_server" />

This cell allows the user to **change the type of Minecraft server** running in Colab.

:::caution
Changing the server software **will delete the current server**.
:::

**Recommendations before running the cell:**

* Make a **backup of your worlds and configurations** if you want to keep them.
* Make sure **no players are connected** to the server before continuing.

:::note
The configuration is the same as explained in [Getting started](/minecolab/guide/getting-started/getting-started) and [Tunnels](/minecolab/guide/tunnel/tunnels).
:::

## Safely Change Server Software

<img src="/assets/images/celdas/safely_server.png" alt="safely_server" />

This cell allows the user to **change or update the Minecraft server version** running in Colab **without deleting the current server**.

**Recommendations before running the cell:**

* Make a **backup of your worlds and configurations** for safety.
* Make sure **no players are connected** to the server before continuing.

:::note
The configuration is the same as explained in
[Getting started](/minecolab/guide/getting-started/getting-started) and [Tunnels](/minecolab/guide/tunnel/tunnels)
:::
