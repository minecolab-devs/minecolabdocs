:::caution
To use the token with TCP tunnels, you need to add a valid credit card as part of the identity verification process.  
Ngrok requires this step to prevent abuse and ensure secure access to TCP services.
:::

## Setting up Ngrok with MineColab

When selecting Ngrok as your tunnel provider, follow these steps to configure it correctly:

### Enter your authentication token

When you select **Ngrok** in MineColab, you will be asked for an **authentication token**. You will see a message like this:

```log
[ LOG ] Chosen Tunnel service: Ngrok
[ LOG ] Tunnel Settings:
[ LOG ] Get your authtoken from https://dashboard.ngrok.com/get-started/your-authtoken
[ WARN ] Remember to paste the whole authtoken at once!
Authtoken:
```

### How to get your token

1. Go to <a href="https://ngrok.com" target="_blank">Ngrok</a> and create an account (if you donâ€™t have one yet).  
2. Once logged in, go to your **Dashboard** and navigate to the **Your Authtoken** section.  
3. Copy the token shown there.

Once you have the token, paste it directly on the next line inside MineColab:

```log
Authtoken:
```
