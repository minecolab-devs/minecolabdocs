## Resourcepacks, Shaderpacks, and Datapacks Information in MineColab

These are **resources that modify visual, graphical, or content aspects** of Minecraft without directly altering the server mechanics. They are installed in specific folders and affect the players' experience.

### What You Should Know:

1. **Server Compatibility:**  
   - Each resource must be compatible with the server's and client's Minecraft version.  
   - Installing an incompatible resource may **cause visual errors or loading issues**.

2. **Compatibility with Most Platforms:**  
   - These resources are compatible with **most server platforms** in MineColab.  
   - Always verify specific compatibility with your server before installing.

3. **Types of Resources:**  
   - **Resourcepacks:** Change textures, sounds, models, and visual elements of the game.  
   - **Shaderpacks:** Enhance game graphics with lighting, shadows, and water effects, but require client support.  
   - **Datapacks:** Add or modify game functions and mechanics (commands, recipes, advancements) without needing mods.

:::note
- **Resourcepacks and shaderpacks** must also be installed on the client for players to see the changes.  
- Always check the **Minecraft version and server type** before using any resource.  
- Consult the **resource documentation** to understand its effects and requirements.
:::
