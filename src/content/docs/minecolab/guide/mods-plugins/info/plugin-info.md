## Plugin Information in MineColab

**Plugins** are extensions installed directly on the **Minecraft server** to add features, automation, or modifications to the game without requiring installation on the client.

---

### What You Should Know:

1. **Server Compatibility:**  
   - Plugins must be compatible with your server version.  
   - Installing an incompatible plugin may **cause errors or operational issues**.

2. **Compatible Server Platforms:**  
- Paper  
- Purpur  
- Folia  
- Banner  
- Arclight  
- Mohist  

3. **What Plugins Can Do:**  
   - Add new gameplay mechanics  
   - Provide additional commands  
   - Manage players  
   - Integrate with other systems (economy, land protection, minigames)

:::note
- Always check the **Minecraft version and server type** before using a plugin.  
- Consult the **plugin documentation** to understand its features and proper usage.
:::
